🚀 HOW TO RUN THE APP:
════════════════════════════════════════════════════════════════════

1. Open Terminal

2. Navigate to project folder:
   cd /Users/sourabhagrawal/Downloads/anshul/jyoti-cards-stock-management-streamlit-app

3. Activate virtual environment:
   source venv/bin/activate

4. Run the app:
   streamlit run app.py

5. Open browser:
   - Desktop: http://localhost:8501
   - Mobile: Use Network URL shown in terminal

📊 HOW TO UPDATE STOCK:
════════════════════════════════════════════════════════════════════

🔧 TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════

❌ App won't start:
   → Make sure you're in the right folder
   → Run: source venv/bin/activate
   → Then: streamlit run app.py

❌ Stock not updating:
   → Click 🔄 in the app
   → Or restart the app
   → Make sure Excel file is saved

❌ Image not showing:
   → Check filename = item number
   → Must be in images/ folder
   → Supported: .jpg, .jpeg, .png
